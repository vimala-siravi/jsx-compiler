# JSX Compiler 
This project is about building a JSX Live Compiler as a React Component

## Getting Started
Clone the repo
```
https://github.com/vimala95/jsx-compiler.git
```

Install the dependencies
```
npm install
```

Run the project 
```
npm start
```
